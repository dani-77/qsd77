# qsd77

qsd77 is cli tool built on go for quickshell-d77.

## How to

```
git clone https://github.com/dani-77/qsd77.git ~/qsd77

cd ~/qsd77

go build

sudo install qsd77 /usr/bin/
```

Autocompletions for bash, fish and zsh can be generated:

```
qsd77 completion bash
```

## Usage

```
|command	|	case		|
|---------------------------------------|
|qsd77 launcher | calls launcher	|
|qsd77 locker	| lock screen		|
|qsd77 session	| calls session menu	|
|_______________________________________|


Enjoy
